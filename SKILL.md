# MCP METAR

MCP server for METAR/TAF aviation weather data.

## When to use this skill

Use this skill when you need to:
- Get current weather for airports
- Fetch terminal aerodrome forecasts
- Get aviation weather information

## Tools

- `get_metar` - Fetch current METAR for an airport by ICAO code
- `get_taf` - Fetch TAF forecast for an airport
- `get_airport_weather` - Fetch both METAR and TAF

## Install

```bash
pip install mcp-metar
```
